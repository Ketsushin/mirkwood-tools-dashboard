# Mirkwood World Dashboard

## Installation
Ordner `mirkwood-world-dashboard` nach:
`FoundryVTT/Data/modules/`

Dann Foundry starten → Module aktivieren.

## Nutzung
- Öffnen über: Einstellungen → Configure Settings → Module Settings → "Mirkwood World Dashboard" → "Open Dashboard"
- Oder über den Scene-Controls Button (GM): "World Dashboard"

## Features
- Profile (z.B. Gondor, Wilderland, Rhovanion)
- Regionen pro Profil
- Globale Werte + Profilparameter
- Automatische Ableitungen: Preise, Verfügbarkeit, Schmuggel, NSC-Haltung
- JSON Import/Export
- Scene-Profil-Zuweisung (optional)
